# My blog

Hugo based. [Live here](https://parsecsreach.org)


## deploying updates

* Have the [deployed repo](https://github.com/emilyselwood/emilyselwood.github.io) cloned somewhere 
* Commit the change here (make sure its marked as not draft)
* Run hugo `hugo`
* Delete everything from the deployed repo except the `.git` folder
* Copy the `public/` directory over to deployed repo.
* Commit the deployed repo
* Push the deployed repo
